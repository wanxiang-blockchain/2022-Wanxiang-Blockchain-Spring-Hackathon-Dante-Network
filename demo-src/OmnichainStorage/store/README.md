# store

##  arweae
### Install
```
cd arweave
yarn 
```

### run a special router(example) functions serving for storage(Arweave)
```
node index.js
```

### create order
```
node applyStorage.js orderdata
```

### read data from arweave tx
```
node readArweave.js SYl6y2SUBNu7J89uwgHHRtvmg5xCFXlvvu9lmYuczdk
```
